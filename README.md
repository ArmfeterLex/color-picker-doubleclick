# Color Picker DoubleClick

Простое приложение на C# Windows Forms, позволяющее менять цвет фона формы двойным кликом мыши.

## Функциональность

- Двойной клик левой кнопкой мыши по форме
- Открытие стандартного диалога выбора цвета (ColorDialog)
- Изменение цвета фона формы на выбранный
- Отмена без изменений при закрытии диалога

## Интерфейс

- Форма с возможностью двойного клика в любой области

## Пример работы

1. Двойной клик по форме
2. Выбор цвета в диалоговом окне
3. Цвет фона формы меняется на выбранный

## Куда можно встроить

- В приложения с настраиваемым интерфейсом
- В демонстрацию работы ColorDialog
- В примеры обработки событий мыши
- В инструменты для персонализации

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Событие MouseDoubleClick привязано ко всей форме. Клик по элементам управления внутри формы не обрабатывается.
